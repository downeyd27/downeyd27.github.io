01/16/2015

Enumeration or enum: is a data type consisting of a set of named values called elements, members or enumerators of the type

#map

map { |object| block } → array
map &8594; an_enumerator
The map method is a very handy method for returning a new array with the results of running the block code once for every element in enum. If no block is given, an enumerator is returned.
